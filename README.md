# python_metro_pred
基于Python的广州地铁客流量统计与预测大数据模型

阶段一、使用Python爬虫爬取微博官网数据采集并保存；
阶段二、使用Echart折线图、柱状图对采集的原始数据做可视化统计；
阶段三、对原始数据进行整合、异常数据预处理、数据清洗（去掉节假日、恶劣天气等）；
阶段四、对处理后的数据做可视化统计，方便与原始数据进行比较；
阶段五、使用ARIMA模型预测和检验历史数据中最后7天客流量数据；
阶段六、使用SARIMA模型预测和检验历史数据中最后7天客流量数据；
阶段七、使用ARIMA模型预测未来7天客流量数据；
阶段八、使用SARIMA模型预测未来7天客流量数据；
